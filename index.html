Verwendete Technologien

Angular

Angular ist ein TypeScript Frontend Framework für die Erstellung von plattformübergreifenden Webanwendungen.

Mit Angular in verbindung Verwendete Technologien:

  • HTML5
  • Sass
     CSS3
  • TypeScript JavaScript

Spring Boot

Spring Boot ist eine Spring Erweiterung, die die Konfiguration und Erstellung von eigenständigen Spring Applikationen vereinfacht.

Mit Spring Boot in Verbindung verwendete Technologien:

  • Java
  • Maven

Adobe XD

Adobe XD ist ein Design-Tool zu Erstellung von Wireframes und Mockups von Web-Apps und Mobile Apps

Docker

Docker ist eine Software zur Containervirtualisierung, sie vereinfacht das Deployment von Anwendungen.

Mit hilfe eines Browsers können die Endnutzer des HelpMyCar-Systems auf die App und das Dashboad, die als Angular Apps vom Frontend zur verfügung gestellt werden, zugreifen.

Die Angular Apps besorgen über HTTP die von den Nutzern angefragten Informationen von dem Spring Boot Server, welcher über eine Caruso Dataplace API-Schnittstelle Informationen von vernetzten Fahrzeugen abfragt und dem Frontend bereitstellt.

Die Zusatzfunktion, die Fahrzeuge im Dashboard auf einer Karte zu sehen, wird mithilfe der GoogleMaps-API ermöglicht.
